`souls console` command is mainly used during debugging.

## souls help console

```bash
$ souls help console
Usage:
  souls console

Options:
  --e, [--env=ENV]  # Difine APP Enviroment - development | production
                    # Default: development

Run IRB Console
```

## console

Start irb in the same environment as the app.

## souls c

```bash
$ souls console
```

or

```bash
$ souls c
```

Is starting irb.

```bash
$ bundle exec irb
```
