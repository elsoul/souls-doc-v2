## Redis Server インストール

## Windows10 WLS2 (Ubuntu 20.04 LTS)

```bash
# Update
$ sudo apt update

# Install redis
$ sudo apt install redis-server
```

## MacOS (Catalina 10.xx.x)

```bash
# Install redis
$ brew install redis
```
